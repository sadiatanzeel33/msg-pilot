# Company Handbook

> This file is the HIGHEST AUTHORITY. Rules here override all prompts.

## General Rules
1. All actions must be logged in /Logs
2. No action is taken without a plan in /Plans
3. Sensitive actions require approval via /Pending_Approval → /Approved flow
4. Guessing is forbidden — ask if unsure
5. Professional corporate tone at all times

## Prohibited Without Approval
- Sending payments
- Emailing new contacts
- Public posting (social media, forums)
- Deleting any file
- Accessing banking systems
- Any irreversible action
- Any MCP external action

## Tier Rules
- Current tier limits are absolute
- Exceeding tier authority requires explicit upgrade approval from the human operator

## File Movement Protocol
- New tasks → /Needs_Action
- Plans created → /Plans
- Risky actions → /Pending_Approval
- Human approves → moved to /Approved
- Human rejects → moved to /Rejected
- Completed work → /Done
- All actions → /Logs (immutable)
